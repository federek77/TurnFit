# TurnFit 🏋️‍♂️

![Vista previa de TurnFit](./Capturas/Captura de pantalla 2025-10-24 145820.png)



**TurnFit** es un sistema de reservas para un gimnasio, pensado para facilitar la gestión de clases, turnos y actividades de forma rápida y moderna. Los usuarios pueden registrarse, iniciar sesión y reservar su lugar en clases específicas.

## 🚀 Características principales

- Registro e inicio de sesión de usuarios
- Visualización del calendario de clases
- Reserva de turnos para diferentes actividades
- Integración con Firebase (Auth + Firestore)
- Interfaz responsive y moderna con Bootstrap

## 🛠️ Tecnologías utilizadas

- HTML, CSS y JavaScript
- Bootstrap 5
- Firebase (Authentication y Firestore)
- Visual Studio Code
